Manage HTML, CSS, and Javascript demos inside WordPress!

== Description ==

Managing your HTML demos should be easy. Since WordPress is easy, let's use that to manage our demos!

Every demo post has HTML, CSS, and Javascript boxes with awesome syntax highlighting. The plugin is smart enough to put your demo in the right place and correctly enqueue your CSS and Javascript. You can supply a custom template for your demos in `single-html_demos.php`.

== Installation ==

Upload the HTML Demos plugin to your WordPress site, activate it, make a demo.

You may want to create a custom template for your demos in `single-html_demos.php`.
